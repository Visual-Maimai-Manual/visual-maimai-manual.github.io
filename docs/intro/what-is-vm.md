# 什么是Visual Maimai
Visual Maimai（简称VM）是由[过氧乙酸不似醋](https://space.bilibili.com/522939904/)开发的一款基于Unity的可视舞萌DX自制谱创作软件，于2024年09月17日正式发行。
## Visual Maimai的优点
Visual Maimai可以让制谱者毋须学习复杂的Simai语言，只通过鼠标与键盘即可进行自制谱创作。  
Visual Maimai把舞萌自制谱的创作难度拉低很多，让初学者通过一段时间的学习后即可写出~~比UTAGE更UTAGE的自制谱~~一个简单的谱面。
## Simai语言是什么
Simai 是一种用于绘制舞萌DX谱面的脚本语言，主要用于定义游戏中的音符位置、类型和时间，使玩家能够在触摸屏上按照音乐节奏进行操作。这种语言广泛用于创建自定义谱面，为舞萌DX提供独特的挑战和体验。（来源于[cnblogs](https://www.cnblogs.com/newzeon/p/18474261)）   
但是由于过于复杂，入门难度较高，所以导致了长期以来制作舞萌自制谱的高难度。Visual Maimai打破了这个现状，把制谱谱面可视化，写完就可以预览，可以让制谱人通过直觉制谱并优化。
